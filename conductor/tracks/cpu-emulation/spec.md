# Specification: CPU Emulation

## Goal

To implement a complete and accurate emulation of the Sharp LR35902 CPU.

## Requirements

1.  8-bit and 16-bit register management.
2.  Complete 8-bit instruction set (opcodes).
3.  CB-prefixed 8-bit instruction set.
4.  Accurate cycle counting for each instruction.
5.  Interrupt handling.
6.  Unit tests for opcodes.
